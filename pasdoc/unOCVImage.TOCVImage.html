function histogram(rangeLow: Integer = 0; rangeHigh: integer = 256; histSize: Integer = 16 ): TArray<TArray<Integer>>;

Calculate the histogram of pixel values of all channels: 3 for color image, 1 for grayscale. The returned array contains one column for every channel

function map(const mapMatrix: TOcvParamMat; const outWidth: Integer = -1; const outHeight: Integer = -1): TOCVImage;

Transform the entire image to another image using a mapping matrix (homography). If out width and height are set to default, the output image will be 4 times greater.
